# 🚀 デスクトップ起動設定ガイド

## 📋 概要

AI Agent Systemをデスクトップから一撃で起動するための設定ガイドです。

---

## 🎯 構成要件

### 1. **仮想環境の自動有効化**
- `.venv` または `venv` フォルダを自動検出
- 見つかった場合、自動でアクティベート
- 見つからない場合、グローバル環境で実行

### 2. **自動起動**
- `streamlit run smart_voice_agent.py` を実行
- ポート8501で起動
- 自動でブラウザアクセスURLを表示

### 3. **エラーの一時停止**
- 起動失敗時に `pause` でコンソールを保持
- エラー内容の確認が可能
- ユーザーフレンドリーなエラーメッセージ

### 4. **アイコン設定**
- デスクトップショートカットの作成
- カスタムアイコンの設定方法

---

## 🚀 使用方法

### **方法1: バッチファイル直接実行**

1. **バッチファイル実行**:
   ```cmd
   C:\Users\GALLE\CascadeProjects\ai_agent_gui\start_ai.bat
   ```

2. **デスクトップにショートカット作成**:
   - `start_ai.bat` を右クリック
   - 「送る」→「デスクトップ(ショートカットを作成)」

### **方法2: Pythonスクリプトでショートカット作成**

1. **必要ライブラリインストール**:
   ```cmd
   pip install pywin32 winshell
   ```

2. **ショートカット作成スクリプト実行**:
   ```cmd
   python create_desktop_shortcut.py
   ```

---

## 🎨 アイコン設定方法

### **1. アイコンファイルの準備**
- 好みのアイコンをダウンロード（.ico形式）
- プロジェクトフォルダに `ai_icon.ico` として保存

### **2. おすすめアイコン**
- **AIロボット**: https://www.iconfinder.com/search?q=ai+robot
- **音声マイク**: https://www.iconfinder.com/search?q=microphone
- **脳アイコン**: https://www.iconfinder.com/search?q=brain

### **3. ショートカットのアイコン変更**
1. デスクトップのショートカットを右クリック
2. 「プロパティ」を選択
3. 「ショートカット」タブの「アイコンの変更」
4. アイコンファイルを選択
5. 「OK」で適用

## 🛠️ トラブルシューティング

### **起動しない場合**
1. **Pythonパス**: PythonがPATHに含まれているか確認
2. **仮想環境**: `.venv` フォルダの権限を確認
3. **ライブラリ**: 手動で `pip install streamlit` を実行

### **ショートカットが作成できない場合**
1. **管理者権限**: コマンドプロンプトを管理者として実行
2. **ライブラリ**: `pip install pywin32 winshell` を実行
3. **手動作成**: バッチファイルを手動でデスクトップにコピー

### **アイコンが表示されない場合**
1. **アイコン形式**: .ico形式であるか確認
2. **ファイルパス**: アイコンファイルのパスを確認
3. **キャッシュ**: アイコンキャッシュをクリア

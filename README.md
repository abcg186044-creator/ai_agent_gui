# テックくん - 成長する親友AIエージェント

技術に詳しい親友のエンジニア「テックくん」と一緒にアプリを作ろう！最新の技術情報を検索して、実際に動くコードを書き、ローカルに保存するまでを友達のような対話で実現します。**使えば使うほど、あなたのことを理解してくれる「成長するAI」**です！

## 🚀 セットアップ

### 1. 依存ライブラリのインストール

```bash
pip install -r requirements.txt
```

### 2. Ollamaのセットアップ

まずOllamaをインストールし、llama3.1モデルをダウンロード：

```bash
# Ollamaのインストール（Windows）
# https://ollama.com/download からインストーラをダウンロード

# llama3.1モデルのダウンロード
ollama pull llama3.1
```

### 3. アプリケーションの起動

```bash
streamlit run app.py
```

## 🎯 主な機能

### 🔍 Web検索
- DuckDuckGo検索で最新の技術情報やニュースを取得
- 「最新のPythonライブラリを教えて」「Reactの新しい機能について調べて」など

### 📝 ファイル作成
- AIが自動でコードやテキストファイルを作成
- 「簡単なWebアプリのコードを書いて」「この処理のPythonコードを作って」など

### 🐍 コード実行
- Pythonコードをリアルタイムで実行・テスト
- 「このコードを実行して結果を見て」「このアルゴリズムを試して」など

### 💬 会話履歴
- 過去の会話文脈を記憶
- 自然な対話の継続

## 🌟 新機能：パーソナライズ機能

### 🧠 長期記憶
- **会話の永続化**: `memory_db.json`にすべての会話を保存
- **アプリ再起動時も記憶保持**: 前回の続きから自然に対話再開
- **スマート記憶管理**: 最新100件の会話を自動で保持

### 👤 ユーザープロファイル自動生成
- **OS情報の自動検出**: 「Windowsで」「Macで」といった会話から自動で学習
- **技術スタックの分析**: Python、JavaScript、Reactなどの使用技術を記憶
- **好みの学習**: 「〜が好き」「〜を学びたい」といった表現からユーザーの興味を理解
- **プロファイル保存**: `user_profile.txt`に学習した情報を保存

### 🌱 成長する相棒
- **パーソナライズされた応答**: 「君の好みは覚えたよ！」「前回のプロジェクトの続きだね」
- **文脈理解**: 過去の会話を踏まえたより自然な対話
- **レベル調整**: ユーザーの技術レベルに合わせて説明の詳細度を自動調整

## 🛠️ 技術構成

- **LLM**: langchain-ollama + llama3.1（ローカル実行）
- **UI**: Streamlit（チャットインターフェース）
- **メモリ**: StreamlitChatMessageHistory + ConversationBufferMemory + 長期記憶（JSON）
- **検索**: DuckDuckGo Search
- **エージェント**: ReAct（思考→行動→結果のサイクル）
- **パーソナライズ**: ユーザープロファイル自動抽出 + 文脈理解

## 📁 プロジェクト構成

```
ai_agent_gui/
├── app.py              # メインアプリケーション（パーソナライズ機能搭載）
├── requirements.txt    # 依存ライブラリ
├── README.md          # 説明書
├── memory_db.json      # 長期記憶データベース（自動生成）
└── user_profile.txt    # ユーザープロファイル（自動生成）
```

## 🎨 パーソナライズ機能の特徴

### リアルタイム思考プロセス表示
AIが今「検索中」なのか「ファイル作成中」なのかを可視化！思考プロセスを展開して詳細を見ることもできます。

### 学習機能
- **自動プロファイル更新**: 会話の中から自然にユーザー情報を抽出
- **文脈理解**: 過去の会話履歴を踏まえた応答生成
- **進化するAI**: 使うたびにユーザーを深く理解

### 友達のような対話
堅苦しいAIではなく、一緒に開発を楽しむ親友のような存在。技術的なことでも気軽に相談できます。

### 実用的なツール統合
検索、ファイル作成、コード実行まで、開発に必要な基本的な機能をすべて統合。

## 🚨 注意事項

- Ollamaがバックグラウンドで実行されている必要があります
- インターネット接続が必要です（検索機能のため）
- ファイル作成時は適切な権限が必要です
- 初回起動時に`memory_db.json`と`user_profile.txt`が自動作成されます
